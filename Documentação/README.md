## 🎯 Fluxo de Trabalho

1. **Você solicita:** "Implementa Tarefa 1.1 (mobile UX)"
2. **Edição local:** Arquivos em `Public/index.html`
3. **Avaliação:** Você testa em navegadores
4. **Aprovação:** Aprova ou pede ajustes
5. **Deploy:** `git push origin main` → Netlify auto-deploys
6. **Documentação:** Achados, métricas, screenshots salvos em subpasta apropriada
